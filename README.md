# Pokemon
这是一个关于破壳萌工具的后端接口部分。

# 上手指南
以下指南将帮助您在本地机器上安装、运行、测试此项目。
## 使用要求
* Java 1.8+
> 项目基于Java开发，要在本机上运行该项目，您应至少拥有1.8+ 的Java JDK。
* Maven 3.0+
> 推荐使用Maven来管理项目。依赖的第三方服务请参考pom.xml文件。
* MySql 5.0+
> 数据库使用MySql。
